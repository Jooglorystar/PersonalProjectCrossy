# PersonalProjectCrossy
  
심화주차 개인과제 '길건너 친구들'을 모티브로 삼은 게임입니다.  
조작은 방향키만 이용하여 이동만 있는 단순한 게임입니다.  

다가오는 차들을 피해 끝의 빨간 라인에 도달하면 되는 게임입니다.  
차에 닿을 시 게임오버되며, 차도 위에 있는 동전을 먹어 점수를 올릴 수 있습니다.
